# nodejs-web-lit-template

## Description
A template for the lit framework.

## Tech stack
- npm
- lit

## Docker stack
- node

## To run
`sudo ./install.sh -u`
- [Availble here](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
